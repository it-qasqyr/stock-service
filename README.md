# Coding Assignment for Payconiq

## Instructions

To run locally the application you have to build it:

```shell
mvn package
```
and then execute the jar:
```shell
java -jar target/stocks-assignment-1.0-SNAPSHOT.jar
```

You can run the tests with:
```shell
mvn test
```
